🔥 Firebase Setup (5 minutes)
  1. Go to console.firebase.google.com
  2. Click "Add project" → name it "rangu-portal" → Create
  3. Click "Web" (</>) icon → register app → copy the config
  4. Go to Authentication → Sign-in method → Email/Password → Enable
  5. Go to Firestore Database → Create database → Start in test mode
  6. Paste your config into signup.html and login.html where it says YOUR_API_KEY
Firebase free tier: 10,000 signups/month, 1GB database — more than enough.
